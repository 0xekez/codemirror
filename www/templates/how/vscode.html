VS Code Installation

{{ end }} {{ define "body" }}
  1. Download the codemirror.vsix file by clicking the button, and place it somewhere you'll remember. Download
  2. Open VS Code and then open the command palette.
    • Cmd/Ctrl + Shift + P
    • View -> Command Palette...
  3. Type vsix and run this command: 
    Extensions: Install from VSIX...
  4. Select the codemirror.vsix file you downloaded in step 1.

Usage

These commands are accessible via the command palette.

  • Start mirroring command: 
    chmod4.mirror: Create New Session
  • Stop mirroring command: 
    chmod4.mirror: Close Session
